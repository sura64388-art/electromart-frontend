# Smart Electrical Management System ⚡

This is a frontend application built using React and Vite.

## Live Demo
https://electrical-frontend.vercel.app

## Technologies Used
- React
- Vite
- Tailwind CSS
- Node.js
- Express
- MongoDB

## Features
- User login system
- Dashboard for electrical management
- Product and stock management
- Responsive UI

## Deployment
Frontend deployed using Vercel.
